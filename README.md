## Iframe

![iframe](https://raw.githubusercontent.com/tomav/mozaik-ext-iframe/master/preview/iframe.iframe.png)

### parameters

key        | required | description
-----------|----------|----------------------------------------------------
`title`    | no       | *Title of the widget*
`url`      | yes      | *iFrame URL*

### usage

```javascript
{
  type: 'iframe.iframe',
  url: 'http://myurl.com/page.html',
  columns: 1, rows: 1, x: 0, y: 0
}
```
